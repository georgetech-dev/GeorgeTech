Privacy Policy for FieldHub Extractor

Effective date: 1 July 2026

FieldHub Extractor is a Chrome extension provided by FieldHub.

This Privacy Policy explains what data FieldHub Extractor accesses, how that data is used, and how users can contact us.

1. Purpose of the extension

FieldHub Extractor allows authorised FieldHub users to extract supported job information from approved job-management systems and create FieldHub-compatible outputs, such as calendar files.

Access to the extension is controlled by FieldHub login and company-level permissions.

2. Data the extension uses

FieldHub Extractor may use the following data:

  • FieldHub login email address
  • FieldHub authentication session/token
  • Company permission settings
  • The currently active job-management page, only when the user clicks an authorised extractor feature
  • Visible job information from that page, such as customer name, job reference, appointment date/time, address, phone number, job notes, and problem description
  • Locally generated calendar file data

The extension does not continuously monitor browsing activity. It does not read all websites automatically. It only reads the active tab when the user chooses to run an authorised extractor feature.

3. How the data is used

FieldHub Extractor uses data for the following purposes:

  • To log the user in to FieldHub
  • To check whether the user’s company is authorised to use the extension
  • To check which extractor features are enabled for that company
  • To read the active job-management page when the user clicks an authorised extractor button
  • To generate FieldHub-compatible outputs, such as calendar files

4. Data storage

The extension stores the user’s FieldHub login session locally in Chrome extension storage so the user does not need to log in every time the popup is opened.

Generated output files, such as calendar files, are downloaded to the user’s computer.

The current version of FieldHub Extractor uses extracted job data locally to generate the requested output. It does not sell, rent, or use extracted job data for advertising.

5. Data sharing

FieldHub Extractor does not sell user data.

FieldHub Extractor does not use user data for advertising.

FieldHub Extractor does not transfer extracted job data to data brokers or advertising networks.

The extension communicates with the FieldHub backend only for authentication, company permission checks, and authorised FieldHub functionality.

6. Remote code

FieldHub Extractor does not execute remotely hosted code.

All extension logic is included in the submitted Chrome extension package.

The extension may communicate with the FieldHub backend for login and permission checks, but it does not download or run remote JavaScript.

7. Permissions used by the extension

  • activeTab: Used to read the currently active job-management page only after the user clicks an authorised extractor button.
  • scripting: Used to run the packaged extractor code on the active tab after the user clicks an authorised extractor button.
  • downloads: Used to save generated output files, such as calendar files, to the user’s computer.
  • storage: Used to save the user’s FieldHub login session locally in the browser extension.
  • host permissions: Used to communicate with the FieldHub backend for authentication and company permission checks.

8. Limited Use disclosure

FieldHub Extractor’s use of data is limited to providing and improving the extension’s stated purpose.

The extension only uses data for user-facing features described in the Chrome Web Store listing and this Privacy Policy.

FieldHub Extractor does not sell data, use data for advertising, or use data for unrelated purposes.

9. Security

We take reasonable steps to protect data used by FieldHub Extractor.

Users should only install FieldHub Extractor from the official Chrome Web Store listing and should keep Chrome up to date.

10. User control

Users can log out of FieldHub Extractor at any time using the Log out button in the extension.

Users can remove the extension from Chrome at any time.

Company access and feature permissions are controlled by FieldHub administrators.

11. Contact

For questions about this Privacy Policy or FieldHub Extractor, contact:

support@fieldhub.uk

Support page: https://www.fieldhub.uk/support

12. Changes to this policy

We may update this Privacy Policy from time to time.

If we make material changes, we will update the effective date above and publish the revised policy on our website.
