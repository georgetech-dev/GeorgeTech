Privacy Policy for GeorgeTech Extractor

Effective date: 3 July 2026

GeorgeTech Extractor is a Chrome extension provided by GeorgeTech.

This Privacy Policy explains what data GeorgeTech Extractor accesses, how that data is used, and how users can contact us.

1. Purpose of the extension

GeorgeTech Extractor helps authorised business users create documents and files from supported field service job pages.

The extension can generate PDF job sheets, email PDF copies, and create iCal calendar files from supported job-management systems.

Access to the extension is controlled by GeorgeTech login and company-level permissions through GeorgeTech CRM.

2. Supported functionality

GeorgeTech Extractor currently supports actions including:

  • Generating PDF job sheets from completed Resolving Sites visits.
  • Emailing PDF job sheets to saved recipient email addresses.
  • Generating iCal calendar files from new or incomplete visits.
  • Importing or extracting supported ServiceBench job information.
  • Checking company-level permissions before allowing access to extension features.

3. Data the extension may process

GeorgeTech Extractor may process job-related information visible on supported field service pages, including:

  • Customer or tenant name, where shown on the job page.
  • Property, site, or job address.
  • Job reference number.
  • Visit ID.
  • Job type.
  • Skill set.
  • Appointment date and time.
  • Attendance, start, completion, or departure times.
  • Engineer notes.
  • Job status or visit outcome.
  • Photos or signatures where included in a generated PDF.
  • Saved PDF recipient email addresses entered by the user.

The extension does not continuously monitor browsing activity. It does not read all websites automatically. It only reads supported job-management pages when the user chooses to run an authorised GeorgeTech Extractor feature.

4. How the data is used

GeorgeTech Extractor uses data for the following purposes:

  • To sign the user in to GeorgeTech Extractor.
  • To check whether the user’s company is authorised to use the extension.
  • To check which extractor features are enabled for that company.
  • To read the active supported job-management page when the user clicks an authorised action.
  • To generate PDF job sheets.
  • To email PDF job sheets to saved recipient email addresses.
  • To generate iCal calendar files.
  • To display the correct extension actions on supported pages.

5. PDF email recipients

Users may save one or more PDF recipient email addresses in the extension settings.

These email addresses are stored locally in the user’s browser extension storage and are used only when the user chooses to email a PDF job sheet.

Multiple recipient email addresses may be separated with semicolons.

6. Data sent to GeorgeTech CRM services

When a user emails a PDF, the PDF and related email details are sent securely to GeorgeTech CRM services so the email can be delivered.

This may include:

  • Recipient email address or addresses.
  • PDF attachment.
  • Company name.
  • Job reference.
  • Visit ID.
  • Job timing details.
  • Engineer notes included in the email body.

GeorgeTech uses Supabase services to authenticate users, check company permissions, and run authorised PDF email processing.

7. Data stored locally by the extension

GeorgeTech Extractor may store the following locally in Chrome extension storage:

  • Signed-in user session information.
  • Company permission settings.
  • Saved PDF recipient email addresses.
  • Basic extension configuration required for the extension to work.

This local data is used to keep the extension working between sessions and to avoid users needing to re-enter settings repeatedly.

8. Generated files

Generated files, such as PDF job sheets and iCal calendar files, may be downloaded to the user’s device.

GeorgeTech Extractor does not intentionally store job page data in the browser after the requested PDF, email, or iCal action has completed.

9. Data sharing

GeorgeTech does not sell user data.

GeorgeTech does not use user data for advertising.

GeorgeTech does not transfer extracted job data to data brokers or advertising networks.

GeorgeTech may process information through service providers required to operate the extension, including:

  • Supabase, for authentication, company permission checks, and Edge Function processing.
  • SMTP or email providers, for sending requested PDF emails.
  • Supported field service platforms, where the user is already logged in and viewing job information.

Information is shared only as needed to provide the requested extension functionality.

10. Remote code

GeorgeTech Extractor does not execute remotely hosted code.

All extension logic is included in the submitted Chrome extension package.

The extension may communicate with GeorgeTech CRM services for authentication, permission checks, and authorised email processing, but it does not download or run remote JavaScript.

11. Permissions used by the extension

  • activeTab: Used so the user can run an action on the currently open supported job page, such as generating a PDF or iCal file.
  • scripting: Used to inject packaged GeorgeTech extraction scripts into supported job platform pages only when the user chooses an action.
  • downloads: Used to download generated PDF job sheets and iCal calendar files to the user’s device.
  • storage: Used to store the user session, company permission settings, and saved PDF email recipient addresses.
  • tabs: Used to identify the active supported job page and open support or email links when requested by the user.
  • host permissions: Used to access supported ServiceBench pages, Resolving Sites pages, and GeorgeTech CRM/Supabase services required for authentication, permission checks, and authorised email processing.

12. Limited Use disclosure

GeorgeTech Extractor’s use of data is limited to providing and improving the extension’s stated purpose.

The extension only uses data for user-facing features described in the Chrome Web Store listing and this Privacy Policy.

GeorgeTech Extractor does not sell data, use data for advertising, or use data for unrelated purposes.

13. Retention

Saved recipient email addresses remain stored locally until the user updates or clears them.

Authentication and permission data may remain stored locally while the user stays signed in.

Email delivery providers and platform infrastructure may retain operational logs for security, troubleshooting, or delivery purposes.

14. Security

GeorgeTech takes reasonable steps to protect information processed through the extension.

Access to extension features is controlled by signed-in user authentication and company-level permissions.

Users should only install GeorgeTech Extractor from the official Chrome Web Store listing and should keep Chrome up to date.

15. User control

Users can:

  • Choose when to generate PDFs, email PDFs, or create iCal files.
  • Update or remove saved PDF recipient email addresses in extension settings.
  • Log out of GeorgeTech Extractor at any time using the Log out button.
  • Remove the extension from Chrome at any time.
  • Contact GeorgeTech for support or privacy questions.

Company access and feature permissions are controlled by GeorgeTech administrators.
